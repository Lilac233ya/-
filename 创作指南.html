01

行情分析 / 实盘复盘

用户最想看
用户要什么不要"肯定涨/跌"，要的是带数据、带逻辑、带风控的完整交易拆解。用户特别想看失败案例的复盘——亏钱不丢人，亏了不知道为什么才丢人。
"不是那种'肯定涨或肯定跌'，而是带图表、带数据、带失败案例的硬核复盘"
— @小梦一场
爆款模板：实盘复盘帖
① 开单背景（什么时间、什么行情下）
② 操作记录（币种 / 方向 / 杠杆 / 止损 / 止盈 / 仓位比）
③ 结果（赚了多少 or 亏了多少，附截图）
④ 复盘分析（哪里判断对了，哪里错了，为什么）
⑤ 教训总结（下次遇到同样情况怎么做）
加分项：附K线标注图、资金费率截图、持仓量变化
创作雷区 — 用户最反感的写法
· 纯喊单："这个币必涨，目标看xxx"——没有逻辑支撑，用户直接划走
· 只有结论没有过程："看多/看空"一句话就没了
· 只晒赚不晒亏：容易制造焦虑，用户明确表示不喜欢
小技巧 — 让用户感受到真实
截图尽量带时间戳，让用户知道你是真实开的单。大胆分享亏损单比只晒盈利更能赢得信任。每篇复盘末尾加一句"欢迎指出我的盲点"，能显著提升评论区互动。
02

新手科普 / 入门教程

用户需求 #2
用户要什么大白话、有截图、一步步教。很多用户"玩了2年才知道挂单比吃单省手续费"——基础知识的市场缺口远比你想的大。
"最少多少U可以玩，怎么调倍数和杠杆"
— @武子4658
爆款模板：保姆级教程
① 标题直接说"解决什么问题"（如：100U怎么玩合约？）
② 先说核心公式/原理（用具体数字举例，别用抽象概念）
③ 分步操作（'打开APP → 点哪里 → 填什么'，最好配截图）
④ 踩坑提醒（新手必犯的3个错误）
⑤ 实操建议（先用5U练手，别第一单就认真）
分层思路：L1 怎么买卖 → L2 合约/网格 → L3 组合保证金/链上操作
写作诀窍 — 怎么写新手不会觉得难
每个专业术语第一次出现时必须用一句大白话解释。比如"逐仓模式（就是爆仓只亏这一单，不会连累你全部资金）"。用户不是笨，是没人教。
03

每日快讯 / 早报

用户需求 #3
用户要什么3分钟获取全球要闻，不是纯搬运，每条新闻都要配一句话解读"这对币价意味着什么"。
"突然插针都不知道为啥，突然拉涨，突然大跌，跟个白痴似的"
— @收割
爆款模板：星球早报
① 宏观面（美联储/政策/地缘，1-2句话说清楚影响）
② 市场数据（BTC/ETH价格、爆仓量、资金费率、稳定币市值）
③ 热点事件（2-3条，每条配一句解读）
④ 今日关注（重要数据公布时间、项目事件）
发布时间：比如每日 8:00 前，适合通勤的宝子刷帖
创作雷区
· 纯截图搬运无解读——用户说"大家都能看到，价值不大"
· 新闻过多不筛选——3-5条最佳，只留真正影响币价的
04

链上数据解读

用户需求 #4
用户要什么不要干巴巴的数字，要把数据"翻译"成普通人能用的信号。"这个数据说明什么，我该怎么做"。
"能不能每周整理一份'聪明钱持仓异动'？"
— @2025起飞
爆款模板：链上数据周报
① 稳定币流向（场内外资金净流入/出 + 一句话信号判断）
② 聪明钱动向（胜率>65%地址近期买卖TOP3 + 表格）
③ 巨鲸预警（单笔>100万美元转账，说清"这意味着什么"）
④ 信号总结（绿灯/黄灯/红灯，直观呈现）
数据源参考：Arkham / Nansen / Defillama / CoinGlass / Dune
写作诀窍 — 数据帖怎么写才不无聊
每个数据后面必须跟一句"信号判断"。不要写"稳定币市值2087亿"就完了，而是"较昨日+1.4亿，资金小幅净流入，中性偏多"。数据是载体，信号才是价值。
05

深度研报 / 项目测评

用户需求 #5
用户要什么不吹不黑，帮用户判断"这个项目值不值得投"。用户明确反感软文和无脑吹，希望看到风险和机会一起讲。
"让我们为了发展前景投资而不是为了热度而投资"
— @酷酷游水的小明
爆款模板：项目测评
① 一句话说清"这个项目干什么的"
② 团队背景（谁做的，融过多少钱，谁投的）
③ 代币模型拆解（分配比例 + 解锁时间表 + 抛压预警）
④ 真实数据（TVL、用户量、收入——是真的还是刷的）
⑤ 风险清单（至少列3条风险，让用户自己判断）
⑥ 结论（不要"DYOR"一句带过，给出你的真实看法）
重点赛道：AI+Web3 / RWA / Layer2 / DeFi / 新公链
创作雷区
· "XX币必火！"式软文——用户一眼就能看出来
· 只写好的不写风险——用户明确说"不吹不黑"
· 堆砌抽象概念无数据支撑——没有TVL、收入等硬指标的测评缺乏说服力
06

空投教程 / 交互攻略

用户需求 #6
用户要什么保姆级步骤 + 风险前置 + 成本透明。用户最怕的不是操作复杂，而是不知道哪个链接是真的、哪个是钓鱼的。
"手动刷空投效率太低，希望能有更系统化的工具推荐"
— @Jim
爆款模板：空投教程
① 成本和风险先说清楚（Gas费多少，有什么风险，有没有可能零回报）
② 分步操作（每步配官方链接，标注"只认这个域名"）
③ 省Gas技巧（什么时间段最便宜）
④ 跨周策略（提醒用户每周交互一次，说明为什么）
⑤ 安全提醒（不要点任何"空投领取"链接，交互后检查授权）
写作诀窍 — 安全感是空投教程的核心竞争力
很多创作者写空投教程只写操作，但用户最担心的是"这个链接安全吗"。在步骤前放一个"成本与风险"表格，用户会觉得你是真正在帮他，而不是在利用他。
07

安全科普 / 避坑指南

用户需求 #7
用户要什么教用户自己查、自己防，而不是等到被骗了才知道。尤其是合约安全检测、貔貅币识别、授权管理这些实操层的内容。
"代币的合约就像股票的财报，底层出了问题再怎么叙事也没用"
— @JAY杰
爆款模板：安全自查指南
① 工具准备（推荐2-3个免费检测工具，配链接）
② 分步教查（合约验证 → 评分检查 → 深度检测 → 交易真实性）
③ 每个步骤配"即刻跑"的红线标准
④ 5秒决策流程图（给用户一个快速判断的框架）
推荐工具：TokenSniffer / GoPlus / DEXScreener / Revoke.cash
写作诀窍 — 安全帖的流量密码
用真实诈骗案例开头（'昨天群里又有人被骗了20万'），再拆解骗术原理，最后教用户怎么防。有故事的安全帖比纯工具教程阅读量高3倍。
08

真实故事 / 社区互动

情感连接
用户要什么有温度的真实经历，不是冰冷的技术分析。用户说"读45分钟还有10分钟休息呢"——星球需要"课间休息"式的内容。
"冰冰冷冷的很机械化，读45分钟还有10分钟休息呢，总不能180分钟都在上课吧"
— @[Miku]
爆款模板：真实故事帖
① 先说结论（"我从5000U做到5万再到归零"）
② 时间线叙述（从入场 → 巅峰 → 崩盘 → 反思）
③ 情感细节（"那天晚上请了三桌饭""盯着爆仓"）
④ 教训清单（用数字总结，比如"学到的6件事"）
⑤ 开放评论（"你的经历是什么？匿名也行"）
特别适合：爆仓复盘、踩坑经历、小白成长日记、周期复盘
写作诀窍 — 怎么写才有"人味"
写你真实的想法，不要用"市场拉升"而是"那天看着绿色数字往上跳，心跳加速"。用户说"比起冰冷的数据和研报，有温度的经历更能给人力量"。真诚比专业更重要。

全板块通用 — 用户最反感的6个雷区

必读
雷区1：无脑喊单
纯点位+"必涨/必跌"，没有逻辑支撑
雷区2：AI水文
复制粘贴、千篇一律、没有个人观点
雷区3：引流广告
拉群返佣、带单诱导、私域引流
雷区4：只晒赚不晒亏
制造焦虑，用户最反感"永赚"人设
雷区5：情绪煽动
标题党、制造恐慌、过度多空对立
雷区6：虚假实盘
实盘不开、截图作假、夸大收益率
用户原话总结："看完没收获"是用户对低质内容的统一评价。写帖前问自己一个问题："别人看完我这篇，能带走什么？"如果答案是"什么都没有"，那就别发。
